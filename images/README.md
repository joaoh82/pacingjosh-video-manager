# Screenshots

## How to Add Screenshots

To add screenshots to this directory:

1. Open the Video Manager application in your browser
2. Take screenshots of:
   - **main-screen.png** - Main video grid with thumbnails and filters
   - **video-grid.png** - Video grid showing multiple videos
   - **video-modal.png** - Video player modal with metadata
   - **settings.png** - Settings page

3. Save the screenshots to this directory with the names above

### Tips for Good Screenshots

- Use full browser window (not just a section)
- Include some sample videos to show the interface in action
- Use light mode or dark mode consistently
- Crop out any personal information or sensitive data
- Recommended size: 1920x1080 or similar 16:9 ratio
